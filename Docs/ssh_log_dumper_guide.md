# SSH Log Dumper 使用教學

## 工具介紹
`SSHLogDumper.exe` 是一個基於 SSH/SFTP 通訊協定的 Log 管理與下載工具。主要用於與測試機台 (預設 IP: `192.168.100.1`) 連線，進行工單 (WO) 資料夾的建立、刪除以及 Log 檔案的同步下載。

## 放置位置
- 執行檔位於：`rawlogs/SSHLogDumper.exe`

## 使用步驟

### 1. 啟動程式與設定連線
- 點擊執行 `SSHLogDumper.exe`。
- 在上方的 **Remote Config** 區域，確認連線參數：
  - **HOST**: 預設為 `192.168.100.1` (若有更改請自行修正)。
  - **PATH**: 預設目標目錄為 `/run/media/nvme0n1p1`。
- 點擊 **「🔍 掃描遠端」**。
  - 左上方指示燈為綠色代表連線成功，並會列出目前遠端存放的工單 (WO) 目錄。
  - 若連線失敗，會顯示紅色指示燈並跳出錯誤提示，請確認設備網路或 SSH 設定 (是否已加入 SSH key)。

### 2. 工單 (WO) 目錄管理
在列表中可以進行目錄的管理：
- **新增 WO**：點擊 **「+ 新增 WO」**，輸入標準格式的工單號碼（例如：`5101-260129012`）。程式會自動在遠端建立對應目錄。
- **刪除 WO**：從列表中選擇一個工單，點擊 **「− 刪除 WO」**。系統會跳出二次確認視窗，確認後將會遞迴刪除遠端該目錄及其所有內容。（請小心使用，刪除後無法復原）

### 3. 設定下載路徑
- 在中方的 **Transfer Config** 區域，點擊 **「...」** 選擇你要將 Log 下載到本機的存放位置（LOCAL）。

### 4. 同步下載 Log
- 從上方列表中點選一個你想要下載的工單 (WO) 資料夾。
- 點擊下方藍色按鈕 **「⬇ 開始同步下載」**。
- 程式會掃描遠端檔案並開始下載：
  - **增量同步**：若本地端已經有相同大小的檔案，會自動略過 (Skip)，節省傳輸時間。
  - **進度顯示**：下方狀態列會顯示複製進度及檔案統計 (複製、略過、失敗)。
  - **中止下載**：在下載過程中，按鈕會變成紅色 **「⏹ 停止」**，點擊可隨時中斷傳輸任務。

## 注意事項
1. **安全機制**：目錄名稱強制檢查為 `XXXX-XXXXXXXXX` (4碼-9碼) 格式，不符格式無法建立或刪除。
2. **通訊底層**：所有新增/刪除及下載皆透過 SFTP 原生指令完成，以確保避免指令注入風險。
3. **佈景主題**：點擊右上角的「🌙/☀」按鈕可以切換深色/淺色主題。

## 疑難排解
- 若發生連線 Timeout，請確認本機與 `192.168.100.1` 之間的網路暢通。
- 程式在 `~/.ssh/` 底下自動尋找金鑰 (`id_ed25519`, `id_ecdsa`, `id_rsa`)，如果沒有設定免密碼登入，請先將 SSH 公鑰放入測試機台的 authorized_keys 之中。
